设置标签大小，图片的大小不会变化,同样改变窗口的大小也不会改变图片的小，只改变图片显示的大小；

  1. 设置窗体关闭规则：setDefualtCloseOperation();

  2. 绝对布局：创狗左上角坐标为（0,0），组件的大小不会随着窗口大小的改变而改变；

  3. 流布局：从左向右像水一样流动，遇到障碍物就换行；setLayout(new FlowLayout(FlowLayout.RIGHT,水平间距，垂直间距)),如果里面什么都不写就是默认水平居中对齐；

  4. 边界布局：将容器划分为五个区域（上、下、左、右、中），添加组件要指定区域，否则默认添加到CENTER，而且同一区域组件会互相覆盖，其实容器的默认布局就是边界布局，添加方法：add(组件名,BorderLayout.方向);会随着窗口大小的变化而变化；

  5. 网格布局：new GridLayout(行，列，水平间距，垂直间距),有一个重要的特性：当你没有添加网格布局的大小时或者当你添加的组件超出你设定网格的大小时，系统将自动添加所需的行和列，窗体能按照你的的布局将其组件完全装下；

  6. 网格组（包）布局管理器：优点是很灵活，但缺点是需要些很多的代码设置，同样是以左上角为原点坐标；
